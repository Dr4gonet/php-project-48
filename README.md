### Utility that calculates differences in text files

### Usage
gendiff (-h|--help)

gendiff (-v|--version)

gendiff [--format ]

### Format options:

stylish

plain

json

### Different formats are output by command:

./bin/gendiff --format <option> <path to file 1> <path to file 2>

### Requirements:

PHP version 8.2.5

Composer version 2.5.3
